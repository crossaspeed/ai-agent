执行压测的命令：
在根目录上执行：（PowerShell）

```
$ts=Get-Date -Format yyyyMMdd-HHmmss; $jmx="docs/jmeter/线程组.jmx"; $jtl="docs/jmeter/result-$ts.jtl"; $html="docs/jmeter/report-$ts"; if (Test-Path $html) { Remove-Item $html -Recurse -Force }; jmeter -n -t $jmx -l $jtl -e -o $html; Write-Host "HTML报告: $html/index.html"
```

打开最新报告：（PowerShell）

```
    $latest = Get-ChildItem docs/jmeter -Directory | Where-Object Name -like "report-*" | Sort-Object LastWriteTime -Desc | Select-Object -First 1; Start-Process (Join-Path $latest.FullName "index.html")
```
